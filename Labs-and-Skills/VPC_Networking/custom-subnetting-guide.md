# VPC Custom Subnetting Guide 🌐

This lab focused on designing and launching a custom Amazon VPC. Key tasks included:

## 🔹 Objectives
- Creating a VPC with CIDR block `10.0.0.0/16`
- Launching 2 public subnets and 2 private subnets
- Configuring an internet gateway and route tables
- Deploying EC2 instances in public subnets with security groups
- Launching a web server and verifying connectivity

## 🧠 What I Learned
- How subnetting improves traffic control and isolation
- Role of route tables in directing packet flow
- Best practices for securing VPC resources (SGs, NACLs)
- Real-world use cases for multi-tier architecture

This guided setup helped me understand the foundational building blocks for deploying resilient cloud applications.
